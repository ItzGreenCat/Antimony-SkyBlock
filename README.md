# What can this mod do
- ItemTranslate -> Chinese
- Secret Bot
- Auto Fish
- Auto KillWorm
- Velocity
- GhostBolck
- Macro
- Block and Mobs ESP
- Custom Pet Name
- Custom Item Sound
- Antimony Mod Channel
- Killaura
- Reach
- AutoUse
